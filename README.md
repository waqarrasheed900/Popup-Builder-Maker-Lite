# Popup Builder & Maker Lite

A lightweight WordPress popup plugin with time-based controls.
Works with Kadence theme and all WordPress themes.

## Features
- Site-wide popup
- Time-based display
- HTML and image support
- Responsive design
- No jQuery, no bloat
